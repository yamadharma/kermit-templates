\section{Введение}

На строящемся в Дармштадте (Германия) ускорительном комплексе FAIR
(Facility for Antiproton and Ion Research) создаётся установка CBM
(Com\-pres\-sed Baryonic Matter). Она предназначена для
экспериментального изучения новых состояний ядерной материи,
образующихся в ядро-ядерных соударениях при энергии пучка 8\textdiv 45
ГэВ на нуклон~\cite{CBM}.

Исследование процессов рождения лёгких векторных мезонов и чармония,
распадающихся на лептонную пару, является одной из ключевых задач
эксперимента СВМ. Для идентификация электронов в диапазоне импульсов
0,5\textdiv 10 ГэВ/с в CBM будет использоваться детектор черенковского
излучения RICH (Ring Imaging Cherenkov). В дополнении к RICH-у
электроны с импульсами выше 1,5 ГэВ/с будут идентифицироваться
детектором переходного излучения TRD (Transition Radiation
Detector)~\cite{CBM}.

Идентификация частиц, регистрируемых детектором RICH, включает в себя
процедуру распознавания и реконструкции колец от черенковского
излучения, а также определение их параметров~\cite{L&O}. Указанная
задача серьёзно осложняется высокой множественностью частиц в каждом
событии, ведущей к большому числу перекрывающихся колец в области
фокальной плоскости RICH-а. В этой связи, важно разработать быстрые и
устойчивые алгоритмы реконструкции колец, позволяющие с высокой
точностью оценить их параметры~\cite{CBM}.

Обычно для описания колец черенковского излучения в RICH детекторах в
качестве математической модели используется окружность (см.,
например,~\cite{Agak,CKO,Lester}). В детекторе СВМ RICH плоскость,
детектирующая фотоны от колец черенковского излучения, не совпадает в
точности с фокальной плоскостью сферических зеркал. Кроме того, в
области RICH-а присутствует небольшое магнитное поле от расположенного
перед ним дипольного магнита, которое хоть и незначительно, но
искривляет траектории заряженных частиц. Все это приводит к тому, что
фотоны черенковского излучения от отдельной частицы распределяются на
детекторующей плоскости не по окружности, а скорее по
эллипсу. Проведённый анализ данных, полученных путём моделирования
процесса регистрации электронов в детекторе RICH, показал, что кольца
черенковского излучения достаточно хорошо аппроксимируются эллипсом с
эксцентриситетом $\varepsilon = 0,43$~\cite{L&O}.

В работе \cite{COS} был разработан метод для оценки параметров
эллипса. В его основу положена минимизация функционала ошибок (смотри
ниже раздел 2.1). Для минимизация указанного функционала использовался
пакет MINUIT~\cite{Minuit}. Этот метод в дальнейшем будем обозначать
как MF (Minuit Fitter).

В настоящей работе развит другой подход для оценки параметров эллипса,
который основан на минимизации функционала, предложенного в
работе~\cite{Tau}. Далее будем обозначать этот метод как TF (Taubin
Fitter). Нами будет проведено сравнение этих методов и показано, что
новый метод имеет ряд преимуществ по сравнению с методом MF.

\section{Методы определения параметров эллипса}

\subsection{Метод MF}

В работе \cite{L&O} реализован алгоритм оценки параметров эллипса на
основе подхода, развитого в работе~\cite{COS}; при этом весовые
коэффициенты в функционале ошибок были заданы равными единице.

В работе~\cite{COS} используется Кеплеровское представление уравнения
эллипса:
\begin{equation}
  \label{eq:M.1}
  d_1  + d_2  = 2a,
\end{equation}
где $d_1 = \sqrt {(x - x_{F1} )^2 + (y - y_{F1} )^2 }$ и $d_2 = \sqrt
{(x - x_{F2} )^2 + (y - y_{F2} )^2 }$, а $(x_{F1} ,y_{F1} )$ и
$(x_{F2} ,y_{F2} )$ являются фокусами эллипса. Определяя невязку между
$i-$ой точкой измерения и модельной кривой в виде $\rho _i = d_1
\left( {x_i ,y_i } \right) + d_2 \left( {x_i ,y_i } \right) - 2a$,
минимизируем следующий функционал ошибок:
\begin{equation}
  \label{eq:M.2}
  L(x_{F1} ,y_{F1} ,x_{F2} ,y_{F2} ,a) = \sum\limits_{i = 1}^n {\rho _i^2 },
\end{equation}
где $n$~--- число отсчётов на рассматриваемом кольце черенковского
излучения.

При минимизации данного функционала использовался пакет
MINUIT. Начальные значения для искомых параметров при инициализации
MINUIT-а определялись следующим образом:
\begin{equation}
  \label{eq:M.3}
  \left\{ \begin{aligned}
      &    a = 5; \\ 
      &    x_{F1}  = \frac{1}{n}\sum\limits_{i = 1}^n {x_i }  - \frac{1}{2}a; \\ 
      &    x_{F2}  = \frac{1}{n}\sum\limits_{i = 1}^n {x_i }  + \frac{1}{2}a; \\ 
      &    y_{F1}  = y_{F2}  = \frac{1}{n}\sum\limits_{i = 1}^n {y_i }. 
    \end{aligned}
  \right.
\end{equation}

Вычисленный таким образом вектор параметров ${\bf{F}} = \left( {x_{F1}
    ,y_{F1} ,x_{F2} ,y_{F2} ,a} \right)^T$, отвечающих минимуму
функционала (\ref{eq:M.2}), преобразовывался затем в вектор новых
параметров ${\bf{P}} = \left( {a,{\rm{ }}b,{\rm{ }}x_c ,y_c ,{\rm{
      }}\varphi } \right)^T$:
\begin{equation}
  \label{eq:M.4}
  \left\{ \begin{aligned}
      & a = a; \\ 
      & b = \sqrt {a^2 - \sqrt {(x_{F1} - x_{F2} )^2 + (y_{F1} - y_{F2} )^2 } } ; \\ 
      & x_c = (x_{F1} + x_{F2} )/2; \\ 
      & y_c = (y_{F1} + y_{F2} )/2; \\ 
      & \varphi = \arctan \left( {\frac{{y_{F2} - y_{F1} }}{{x_{F2} - x_{F1} }}} \right). 
    \end{aligned}
  \right.
\end{equation}
где $\varphi$~--- угол наклона большей полуоси эллипса к оси $OX$.

Это преобразование позволяет получить величины большой $a$ и малой $b$
полуосей эллипса, которые нужны для идентификации электронов в RICH
детекторе.

Недостатки рассмотренного метода: 1) использование громоздкого пакета
MINUIT для минимизации функционала ошибок, 2) необходимость задания
хорошего начального приближения для того, чтобы обеспечить сходимость
процесса минимизации~\cite{L&O}.

\subsection{Taubin Fitter}

Возьмём в качестве математической модели для описания результатов
измерений кривую второго порядка $P\left( {x,y} \right) = 0$. В общем
случае, для оценки параметров модели, необходимо минимизировать сумму
квадратов расстояний от измеренных точек до модельной кривой:
\begin{equation}
  \label{eq:T.1}
  {\cal S}\left({x_i ,y_i ,{\bf{\Theta }}}\right) = 
  \sum\limits_{i = 1}^n {d_i^2},
\end{equation}
где $\bf{\Theta}$~--- вектор параметров модели. Для кривой второго
порядка ${\bf{\Theta}} = \left(A,B,C,D,E,F\right)^T$.

В результате минимизации функционала ${\cal S}\left( {x_i ,y_i
    ,{\bf{\Theta }}} \right)$ мы получаем кривую второго порядка
наиболее адекватно отображающая результаты измерений. Но в таком
представлении задача не всегда поддаётся однозначному решению за
разумное время. На практике для преодоления указанной проблемы
накладываются дополнительные условия.

В частности, в работах \cite{Tau,Sam,Zh} для решения указанной
проблемы используется линейная аппроксимация расстояния от измеренной
точки до модельной кривой:
\begin{equation}
  \label{eq:T.2}
  \frac{{\left\| {P\left( {x_i ,y_i ,{\bf{\Theta }}} \right)} \right\|}}
  {{\left\| {\nabla P\left( {x_i ,y_i ,{\bf{\Theta }}} \right)}
      \right\|}}  = d_i  + O\left( {d_i^2 } \right).
\end{equation}
В этом случае функционал (\ref{eq:T.1}) принимает следующий вид:
\begin{equation}
  \label{eq:T.3}
  {\cal P}\left( {x_i ,y_i ,{\bf{\Theta }}} \right) = 
  \sum\limits_{i = 1}^n {\frac{{\left\| {P\left( {x_i ,y_i ,{\bf{\Theta
                  }}}  \right)} \right\|^2 }}
    {{\left\| {\nabla P\left( {x_i ,y_i ,{\bf{\Theta }}} \right)} \right\|^2 }}},
\end{equation}
где градиент функции $P\left( {x,y} \right)$ выражается следующим
образом:
\begin{equation}
  \label{eq:T.3.1}
  \nabla P\left( {x_i ,y_i ,{\bf{\Theta }}} \right) = \begin{pmatrix}
    2 A x_i + B y_i + D \\ 
    2 C y_i + B x_i + E  
  \end{pmatrix}.
\end{equation}

Оценка параметров кривых второго порядка, полученная при
минимизации~(\ref{eq:T.3}), называется взвешенноградиентной
оценкой~\cite{Tau,Sam,Zh}:
\begin{equation}
  \label{eq:T.4}
  {\cal P}\left( {x_i ,y_i ,{\bf{\Theta }}} \right) = \sum\limits_{i = 1}^n {\omega _i \left[ {P\left( {x_i ,y_i ,{\bf{\Theta }}} \right)} \right]^2 },
\end{equation}
где $\omega _i = 1/({{\left\| {\nabla P\left( {x_i ,y_i,{\bf{\Theta
              }}} \right)} \right\|^2
  }})$ % $\omega _i = \frac{1}{{\left\| {\nabla P\left( {x_i ,y_i
% ,{\bf{\Theta }}} \right)} \right\|^2 }}$.

В работе \cite{Tau} предложено заменить функционал (\ref{eq:T.3}) на:
\begin{equation}
  \label{eq:T.5}
  {\cal L}\left( {x_i ,y_i ,{\bf{\Theta }}} \right) = \frac{{\sum\limits_{i = 1}^n {\left[ {P\left( {x_i ,y_i ,{\bf{\Theta }}} \right)} \right]^2 } }}{{\sum\limits_{i = 1}^n {\left\| {\nabla P\left( {x_i ,y_i ,{\bf{\Theta }}} \right)} \right\|^2 } }}.
\end{equation}
Тогда $\omega = 1/\left({{\sum\limits_{i = 1}^n {\left\| {\nabla
            P\left( {x_i ,y_i ,{\bf{\Theta }}} \right)} \right\|^2 }
    }}\right)$ будет общим для всех точек, что значительно упрощает
процедуру минимизации.

Положим $\sum\limits_{i = 1}^n {\left\| {\nabla P\left( {x_i ,y_i
          ,{\bf{\Theta }}} \right)} \right\|^2 } = 1$, тогда выражение
(\ref{eq:T.5}) примет следующий вид:
\begin{equation}
  \label{eq:T.6}
  {\cal L}\left( {x_i ,y_i ,{\bf{\Theta }}} \right) = \sum\limits_{i = 1}^n {\left[ {P\left( {x_i ,y_i ,{\bf{\Theta }}} \right)} \right]^2 }.
\end{equation}
В матричной форме задача минимизации выглядит следующим образом:
\begin{equation}
  \label{eq:T.7}
  {\cal L} = {\bf{\Theta }}^T \,{\bf{M}}\,{\bf{\Theta }} =  > \min,
  \text{ при } {\bf{\Theta }}^T \,{\bf{B}}\,{\bf{\Theta }} = 1,
\end{equation}
где $\sum\limits_{i = 1}^n {\left[ {P\left( {x_i ,y_i ,{\bf{\Theta }}}
      \right)} \right]^2 } = {\bf{\Theta }}^T \,{\bf{M}}\,{\bf{\Theta
  }}$ и $\sum\limits_{i = 1}^n {\left\| {\nabla P\left( {x_i ,y_i
          ,{\bf{\Theta }}} \right)} \right\|^2 } = {\bf{\Theta }}^T
\,{\bf{B}}\,{\bf{\Theta }}$.

Отсюда следует, что
\[ {\bf{M}} = \left[
  \begin{matrix}
    {M_{xxxx} } & {M_{xxxy} } & {M_{xxyy} } & {M_{xxx} } & {M_{xxy} } & {M_{xx} }  \\
    {M_{xxxy} } & {M_{xxyy} } & {M_{xyyy} } & {M_{xxy} } & {M_{xyy} } & {M_{xy} }  \\
    {M_{xxyy} } & {M_{xyyy} } & {M_{yyyy} } & {M_{xyy} } & {M_{yyy} } & {M_{yy} }  \\
    {M_{xxx} } & {M_{xxy} } & {M_{xyy} } & {M_{xx} } & {M_{xy} } & {M_x }  \\
    {M_{xxy} } & {M_{xyy} } & {M_{yyy} } & {M_{xy} } & {M_{yy} } & {M_y }  \\
    {M_{xx} } & {M_{xy} } & {M_{yy} } & {M_x } & {M_y } & N
  \end{matrix}
\right],
\]
\[ {\bf{B}} = \left[
  \begin{matrix}
    {4 \cdot M_{xx} } & {2 \cdot M_{xy} } & 0 & {2 \cdot M_x } & 0 & 0  \\
    {2 \cdot M_{xy} } & {M_{xx}  + M_{yy} } & {2 \cdot M_{xy} } & {M_y } & {M_x } & 0  \\
    0 & {2 \cdot M_{xy} } & {4 \cdot M_{yy} } & 0 & {2 \cdot M_y } & 0  \\
    {2 \cdot M_x } & {M_y } & 0 & N & 0 & 0  \\
    0 & {M_x } & {2 \cdot M_y } & 0 & N & 0  \\
    0 & 0 & 0 & 0 & 0 & 0
  \end{matrix}
\right],
\]
где $M_{xxxx} = \sum\limits_{i = 1}^N {x_i^4 }$, $M_{xyyy} =
\sum\limits_{i = 1}^N {x_i y_i^3 }$, $M_{xy} = \sum\limits_{i = 1}^N
{x_i y_i }$ и т.д.

Задача (\ref{eq:T.7}) решается методом множителей Лагранжа:
\begin{equation}
  \label{eq:T.8}
  {\cal F}\left( {{\bf{\Theta }},\eta } \right) = {\bf{\Theta }}^T \,{\bf{M}}\,{\bf{\Theta }} - \eta  \cdot \left( {{\bf{\Theta }}^T \,{\bf{B}}\,{\bf{\Theta }} - 1} \right).
\end{equation}

Дифференцируя ${\cal F}\left( {{\bf{\Theta }},\eta } \right)$ по
вектору $\bf{\Theta }$ и приравнивая производную к нулю, получим
обобщённую задачу на собственные значения:
\begin{equation}
  \label{eq:T.9}
  {\bf{M}}\,{\bf{\Theta }} - \eta  \cdot {\bf{B}}\,{\bf{\Theta }} = {\bf{0}}.
\end{equation}

В задаче (\ref{eq:T.9}) обобщённый собственный вектор, соответствующий
минимальному положительному обобщённому собственному значению, даёт
минимум функционала (\ref{eq:T.5})~\cite{Tau}. Вектор $\bf{\Theta }$
является вектором параметров кривой второго порядка и легко
преобразуется в вектор параметров ${\bf{P}} = \left( {a,{\rm{
      }}b,{\rm{ }}x_c ,y_c ,{\rm{ }}\varphi } \right)^T$.

\section{Сравнительный анализ методов}

Рассмотренные выше методы оценки параметров эллипса тестировались на
двух выборках.

В первом случае моделировались $10^5$ эллипсов с полуосями $a = 6.2$ и
$b = 5.6$ и центром $x_c = y_c = 0$. Случайным образом генерировалось
число точек на эллипсе, а к координатам каждой точки добавлялась
случайная величина, распределённая по нормальному закону $N(0,\sigma)$
с $\sigma = 0.3$, имитируя измерительную ошибку. Разброс числа точек
на эллипсе и величина ошибки в определении их координат выбирались на
основе априорной информации, полученной в результате моделирования
процесса регистрации электронов детектором СВМ RICH~\cite{CBM}.

На рис. \ref{fig1}--\ref{fig2} представлены результаты тестирования
обоих методов. Как видно из рис.~\ref{fig1}, с помощью метода TF на
вычисления тратится в 5\textdiv 25 раз меньше
времени\footnote{Вычисления проводились на компьютере с процессором
  Intel Core 2 Duo с частотой 2.13 ГГц} по сравнению с методом
MF. Кроме того, оценки параметров, получаемые с помощью TF-метода,
примерно в два раза ближе к истинным в среднем в смысле евклидовой

Во втором случае для тестирования использовались модельные данные,

UrQMD~\cite{UrQMD} моделировался акт первичного ядро-ядерного
взаимодействия, а затем вторичные частицы протаскивались через всю
установку СВМ с помощью программы GEANT~\cite{GEANT}, в том числе, с
учётом регистрации заряженных частиц детектором RICH.

На этих модельных данных оценивалась эффективность процедуры
распознавания колец черенковского излучения, а также точность
определения большой и малой полуосей эллипса.

\section{Заключение}

В работе развит альтернативный подход для оценки параметров колец
черенковского излучения TF, основанный на методе предложенном в
\cite{Tau}. Ниже приведены результаты сравнительного анализа MF и TF
методов:
\begin{itemize}
\item скорость вычислений параметров эллипса методом TF в 5\textdiv 25
  раз превышает скорость вычислений с помощью MF метода
  (рис. \ref{fig1});
\item точность оценки параметров эллипса с помощью TF метода
  приблизительно в два раза превышает точность, достигаемую методом MF
  (рис. \ref{fig2});
\item эффективность процедуры распознавания электронных колец при
  использовании метода TF в среднем на 2.69\% выше по сравнению с
  использованием MF метода (рис. \ref{fig3}).
\end{itemize}

Приведённые результаты наглядно демонстрируют преимущества метода TF
по сравнению с методом MF. Однако следует заметить, что с помощью
метода TF мы получаем параметры кривой второго порядка в общем виде,
которая может не всегда отвечать эллипсу. Проведённая нами проверка
показала, что такие случаи действительно возможны, хотя очень редки. В
тоже время, т.к. в методе MF в качестве модели взят эллипс, то на
выходе он однозначно даёт нам оценки малой и большой полуосей эллипса,
что можно считать преимуществом MF в сравнении с TF методом.

Оба метода включены в программную среду CBM
ROOT~\cite{FairRoot1,FairRoot2} и активно используются участниками
коллаборации СВМ.
